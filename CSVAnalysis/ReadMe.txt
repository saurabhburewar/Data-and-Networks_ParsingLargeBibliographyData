This part consists of modules which analyse the data directly from the CSV files
It creates a distribution of Co-authors similar to "Database and Analysis/CoAuthorPlot" 

To build a plot
**Run plot.py file

1> The system will generate a plot from CoAuthors.csv file and will be saved in a .dot file.
   After obtaining data.dot file we can obtain image by running graphviz command on it in terminal.

2> The multiplicity, node and co-author stats will also be calculated and displyed.
